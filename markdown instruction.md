## Списки

Чтобы добавить ненумерированные списки, необходимо пункты выделить звездочкой (*). например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерированные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
